# Swiggy/Zomato Delivery Analytics
A complete food delivery operational analytics project using Python (Pandas), MySQL, and Excel dashboards to analyze rider performance, delivery efficiency, cancellations, distance-based challenges, and peak-hour operations.

Author: Neethu Raj  
Tools Used: Python, Pandas, MySQL, Microsoft Excel

## Project Overview

This project analyzes food delivery operations using Python (Pandas), SQL, and Excel dashboards. The analysis focuses on delivery efficiency, rider performance, operational KPIs, late deliveries, and customer demand patterns.

---

## Objectives

- Analyze food delivery operations
- Measure rider efficiency and productivity
- Identify peak order demand periods
- Study delivery delays and operational bottlenecks
- Evaluate distance impact on delivery performance

---

## Key Insights

- Peak order demand occurs during afternoon and evening hours.
- Longer delivery distances increase average delivery time.
- Certain riders consistently outperform others in efficiency.
- Late deliveries increase during high-demand periods.
- Rider workload varies across operational periods.

---

## Business Recommendations

- Increase rider allocation during peak hours.
- Monitor long-distance deliveries to reduce delays.
- Improve operational efficiency during busy periods.
- Reward consistently high-performing riders.
